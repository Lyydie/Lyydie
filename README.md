## 🚀 Featured Projects

### 🤖 AI Product & Catalog Engine — *IGAF Technologies / ExportConnect* *(ongoing, 2026)*
Multi-agent system that transforms raw product data into multilingual B2B-ready export sheets. Built with LLMs (GPT/Claude), RAG pipelines, LangChain, FastAPI, React — includes ISO/ESG compliance validation and an anti-greenwashing verification layer.

### 📊 Intelligent Internship Matching System — *Algérie Poste* *(Feb 2026)*
Data mining pipeline for automated candidate-to-department matching and HR decision prediction. Features semantic skill taxonomy (120 categories, 1000+ synonyms, +34% matching accuracy), SVM/Random Forest classifiers (80% accuracy), anomaly detection (Isolation Forest, LOF), and full candidate ranking system. Dataset: ~3,000 profiles.

### 🏆 Multilingual Job Matching Platform — *3rd place, Ministry of Knowledge Economy Hackathon*
Full-stack NLP platform processing 15,000+ job listings using SBERT semantic similarity. Built with Flask backend and a hybrid AI + human validation workflow.

### 🏆 Amanar — *1st place, ENSIA Ideathon 2025*
Gamified mobile app for Saharan tourism combining AI-powered trip recommendations, cultural heritage modules, and a virtual companion. Full product design: market study, BMC, PESTEL, Figma prototype, pitch.

### 🏥 Clinic Appointment System *(deployed)*
Full-stack medical scheduling app: patient booking, admin dashboard, RBAC authentication, real-time availability checks, and automated confirmation emails. Stack: Next.js, Supabase, Tailwind CSS.

### 📱 Sellekni — Peer-to-Peer Task Platform
Mobile app connecting users who need help with everyday tasks to nearby helpers. Built with Flutter (Dart), modular architecture, Cubit state management, SQLite.

### 📈 Semiconductor Sales Forecasting
Time series analysis on 481 monthly observations (1986–2026). SARIMA(1,1,1)(1,0,1)[12] selected via Box-Jenkins methodology (AIC = -1418). 4–6% error on 3-month horizon. Built with R.

### 🛡️ Network Security Tools — *ENSIA Cybersecurity*
Multi-threaded TCP port scanner with SYN scanning (Scapy), banner grabbing, CVSS-inspired risk scoring, and structured report generation (terminal / JSON / HTML). Also built a malicious browser extension simulator (Pomodoro cover) for security awareness education.
